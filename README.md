Logic Circuit Simulator
A C++ application that allows users to design and simulate digital logic circuits using fundamental logic gates and interactive components.

This project is a logic circuit simulator developed using C++ and Object-Oriented Programming (OOP). It enables users to build, edit, and test digital circuits by connecting components such as logic gates, switches, and LEDs.The simulator helps visualize circuit behavior and simplifies the analysis of complex logical systems.

Features
1. Design Mode
-Add logic gates: AND, OR, NOT, NAND, NOR, XOR, XNOR
-Add input/output components:
   .Switches (inputs)
   .LEDs (outputs)
-Connect components via input/output pins
-Label components and connections
-Select, move, and delete components dynamically

2. Simulation Mode
-Simulate circuit behavior in real time
-Toggle switches to change inputs
-Observe outputs through LEDs

File Operations
-Save circuit designs to file
-Load previously saved circuits

Concepts & Technologies
-Language: C++
-Programming Paradigm: Object-Oriented Programming (OOP)
